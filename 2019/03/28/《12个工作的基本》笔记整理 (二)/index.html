《12个工作的基本》笔记整理 (二)

发表于 2019-03-28 | 更新于 2020-07-12 | 分类于 书籍 | 评论数： | 阅读次数：

第五能力：持续学习力（20-40岁）

在必要时学习必要知识的习惯。

缺少该能力将跟不上新形势，知识量无法继续增加。在进入大学之前学校会通过作业来要求我们学习，但到了大学由之前的强制学习变成了自主学习。
缺乏自主学习经验，漫无目的的考取资格证书，最终一事无成。切勿死记硬背需要灵活运用。通过不断的阅读将思维格式化。

学习转移，一件事情成功后，再做第二件事就会变得简单。

养成学习的关键：

  1. 精力集中，下定决心就要执行。
  2. 确定近期目标，督促自己每天持续学习。
  3. 记住的立刻说出来。分享的过程同时也是巩固自己学习的过程。
  4. 养成学习习惯，如固定时段就一定要学习。
  5. 灵活运用，收获成果。学到的东西要灵活运用到工作中。

第六能力：语境理解力（20-50岁）

理解对方的语境是有效沟通的第一步。

缺少该能力将无法成为项目负责人。不能一味的对自己的理论夸夸奇谈，沟通是存同去异的过程，不能一开始就抱有抵触的心态，找出相同处，处理不同的地方。

语境理解力需要：感情控制力、逻辑思考力、表达力。 同对方交谈需要保持良好心态，如果不能控制自己的情绪，是不能和对方产生共鸣的。接着是需要不断的思考去理解对方所想表达的意思。

积累与语境不同的热沟通的经验，不同年龄段、不同专业领域、不同国籍的人频繁交流，语境理解力将自然而然的得到锻炼。代沟会使人难以沟通。想象对方的发言背景，才能体会出语言背后的心情和需求。

整理论点，开会讨论时将需要讨论的点整理出来，这样就不会纠缠于琐碎的讨论。语境理解力最大限度的发挥场合就是谈判。

第七能力：专业构筑力（30-50岁）

有计划的构建自身独特的强项能力

缺少该能力将在40岁后，仍然对任何事都缺乏自信。专业能力需要下定决心，定制非常细致的计划才行。

专业的选择：

  1. 尽量选择已经累积大量经验的领域。
  2. 公司内部是否提供该专业发挥的可能。
  3. 选择较短时间内达到一定水平的领域。
  4. 想象出自己享受该领域工作的模样，确认自己是否会适应这个职务。

构筑专业领域重要的原则是“理论和实践两方面都要平均深化”，要能把理论运用到实际，又能从理论上说明实践，才称得上专业。

记录自己学习、实践过程中所产生的心得，然后整理、归纳。

专家需要知道每天专业相关的报道和新闻，必须了解当下专业的走向。

专家关系网非常重要，可以相互交流信息，还可以通过人际关系网传递私密信息。

第八能力：人脉开拓力（30-60岁）

开拓业务伙伴或信息来源，并与之维持关系的能力。

缺少该能力摘去头衔则一事难成。在你专业领域之外你将难以完成。人脉非常重要，尤其是30岁之后。

主动去见自己想见的人，发展并维持关系。想跟见过一次面的人继续维持关系？短时间内见两次面，可以通过你们的共同好友加深彼此的关系。将自己的好友介绍给第三方是加深关系的有效技巧。这样关系网将不断的壮大，且稳固。

“很少见面的人”能力“经常见面的人”身上得到更多的有用信息。

怀着无私态度来开拓人脉，不要吝啬交际费和饮食费。在二三十岁的时候最好把钱用来自我投资。

在个人主页上介绍自己，停止加班空出时间，将更多的时间留给家人、学习或会客，想别人请教才能不断为自己输入。

第九能力：委任力（30-50岁）

不是领导也需要具备委托别人工作的能力

所有工作都自己承担，无法肩负重任。当你30岁手低下有一两个部下的时候是锻炼委任力的最佳时期，20多岁的时候把工作委派给后辈也能达到热身的状态。

如果一个人不能掌握委任力，就无法肩负重任。因为一个人的能力始终是有限的。

“自己做更快”的恶魔之声，有时为了避免麻烦会将自己完成这工作，因为他人可能做得不够好，还得自己重做一遍，这是恶魔之声就响起了。一开始将工作给后辈做，免不了自己承担失败的后果。

将工作委任给下属比你自己完成更有挑战性，因为你必须将你所理解的解释给你的下属，还要判断出他对你解释理解的程度，当然你还必须有足够的耐心。此外你还要在中途检查工作的进程，如果对方做的仍然不到位，你还得必须加以修改。比起自己做修改别人的东西当然会更难。将工作妥善的交给别人对自己水平要求也是非常高的。

当你将工作托付给别人做，就应该好好的赞赏、慰劳对方。信赖是需要“眼光”的，你需要更够准确的判断出对方的强项、弱项和能力，并且确信他能完成任务。这种“眼光”是从大量的失败锻炼出来的。

需要对工作有一个整体的设计，做出计划安排。

第十能力：商谈力（40-70岁）

提出中肯的建议把决定权留给对方。

缺少该能力：无视对方的价值观，把自己的想法强加于人，一味的说教。
咨询协商的能力，善于倾听并与别人一同思考问题的能力。

掌握该能力需要三步：

  1. 与对方建立彼此信赖的人际关系，这是前提。
  2. 充分认同对方的价值观和特性。
  3. 提供专业的建议，让对方做出决定。

随着经验的积累、专业性的掌握，同时也具备一定的判断力，有些中年人逐渐变得听不进陌生人说话。只想把自己的想法强加给对方，或者匆匆给出了否定的结论。这样自己将被人群孤立，不会有人和你商谈，导致一个人单打独斗，但是许多东西都是靠团队的力量完成的。如果上司只会单方面地强加于人，就不会受到欢迎。跟部下交谈时需要理解对方的想法并且提供适当的建议。

理想的上司是职业顾问型，希望上司能够详细的阐述工作的意思，认可部下的优点，能耐心的提供建议，帮助部下迅速的进步。同时让部下感受到工作的意思，充分发挥自己的能力并不断进步，同过社会认可感受到自我存在的意义。

不可忘记的要点

如果你想掌握基本的咨询技能,我建议去听职业顾问培训讲座。此外，我将在下面介绍一些具备“商谈力”的人的特性。

1.酝酿信赖关系。信赖源自“觉得对方不会对自己不利”的情绪，而这样的情绪源自“对方很理解自己”的认识。如果对方并不试图理解你的处境、价值观、能力、爱好等因素，你是不会对其产生信赖感的。

2.有相同的目标。就咨询而言，咨询双方拥有同一个目标是极其重要的。如果彼此的目标不同，就失去了咨询的前提。以园艺用品专卖店为例，顾客购物的目的是“想把某个品种种在花盆里，让它开满鲜花”, 如果销售员也怀着同样的心情，顾客就会用心倾听他的建议。相反，如果销售员的言行中透露出唯利是图的态度，顾客就根本不想听他说什么。上司与部下的关系也一样，如果上司一味表现出只想提升业绩的态度，他就称不上职业顾问型管理者。只有抱有希望部下(从对方的角度来说就是自身)进步的大目标，才能真正笼络人心。

3.基于专业知识，作出详细说明。你既然拥有该领域的专业知识，就必须作出详细的说明，也就是以简单易懂的语言向部下介绍有哪些选择，这些选择各有哪些优点和缺点，等等。越是需要作出重大决策的工作，这种说明就越重要，如果说明不到位，就是你的责任。

4.最终让对方决定。最后的重要决定不能由自己来作，必须让对方本人基于自身责任作出。咨询的目的正是为了支持对方自己作出决定。如果决定不是出于自己的意愿作出的，过后就很可能轻易变卦，有时还会觉得结论是别人强加给自己的。
商谈力是交流能力的延伸。只要掌握这样的交流风格，你的专业能力就能大放光彩。

第十一能力：传授力（40-70岁）

把自己掌握的知识或技术教给别人的能力

缺失改能力：无法把自己的知识传给后辈或部下。教别人绝不是一件容易的事，优秀的管理者需要具备培养人才的技术。

需要注意：

  1. 说明不足，“看着我学”无法将自己的知识通过语言教给别人，除非是学徒制潜移默化的传给别人。
  2. 滔滔不绝，单方面将个不停，不去管对方有没有听懂，话题也飘忽不定。
  3. 夸夸其谈，洋洋得意的炫耀自己以前干这样那样的事情，如果适可而止对方也会心有敬意，但是过分吹嘘那么适得其反。
  4. 说教，从一开始就抱有批判的态度和对方交流，这样从一开始对方就会抱有抵触的心态。因果关系，种豆得豆。
  5. 比喻过剩，乱打比喻导致对方难以理解。

传授方和受教方容易产生上下级关系，但传授的关系如果是平等的话效果更佳。“一同”思考的平等伙伴关系更能提高学习效果，这种平等的伙伴关系的能力，不仅能提升你的传授力还能拓展人脉，提高交流能力。

能和对方产生共鸣，建立和谐的关系那么传授力就能维持在很高的水准。在对方感到沮丧可以静静的问问“怎么了”反而容易诱使对方一吐不快。
毫无保留是大忌，需要留给对方思考的空间，什么都教给对方那么受教方便一无所获。传授方应该发挥聆听和提问的能力。部下向上司求建议，上司也应该有所保留。如果事事都解答那么部下会依赖上司，从而放弃自己思考。

适当的夸奖和责备，夸奖时要让对方清除自己哪里做的好，最好当众夸奖。而责备就好是一对一，责备应该责备具体的“行为”，而不是批评对方的“品行”和“态度”。

要点：与对方建立和谐的关系，灵活利用提问、聆听等技巧。不要说太多，适当的进行夸奖和责备。

第十二能力：协调力（40-70岁）

调整、中介、推进事物的能力，促使不同利益方达成共识，统一目标。

缺少该能力：无法完成大型工作，该能力需要上述的11种能力。

协调者首先要具备相应领域的专业知识和人脉，然后发现对方的需求，偶尔还要一边教学，一边指定最完善的计划。而且，当涉及多个利益相关者的时，协调者需要居中协调，妥善地加以引导，促使众人达成共识，确定统一目标。

协调者需要在不同领域间协调。一边提供建议，一边引导解决。

构成协调力的10个要素：
1.专业知识，2.丰富的人脉，3.把握对方的需求，4.本质的开朗，5.针对信息收集的日常活动，6.目标的设定，7.达成共识的推进，8.适当的建议，偶尔还可以加以规劝。9.勤勉，为了实现目标不懈的努力，10.为人诚实。

最后

12 种基本能力有着各自的最佳开发时期，一旦过了最佳时期掌握相应能力的难度就会加大，这个时候就需要更多的努力，不要放弃。如果为时已晚那么可以放弃改掉弱点的想法，把精力放在拓展强巷上，借此弥补弱点造成的不足。“在强项上构建自己”。两个必要条件“信息收集”和“数字解读技术”。

不断提升自己的职业能力，但也不要忘了享受生活。
